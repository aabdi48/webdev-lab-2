Course Catalog Description

This course covers the creation of web pages and sites using HTML, CSS, JavaScript, and graphical applications. Networked multimedia distribution technologies are explored. The course focuses on effective website design including layout, user interface, graphics, content flow, site structure, and website management. Students design and create a major multi-page website with cross-linked pages.

Prerequisites: None
Credit: 2-2-3 Semester Hours

Course Outcomes

  • Recognize HTML, CSS, and JavaScript code
  • Select correct markup and code
  • Identify improperly used code
  • Create standards-compliant web pages
  • Deploy pages to GitHub

Course Objectives

  • Explain client-server architecture of the internet
  • Create web pages using HTML, CSS, JavaScript, and jQuery
  • Deploy websites using GitHub and version control
